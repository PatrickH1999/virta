Detailed Description

High-level parallel iteration utilities using OpenMP or serial execution.

This module provides wrappers for one-dimensional and two-dimensional parallel loops, as well as a simple parallel region helper. All functions accept user-provided lambdas in the same design spirit as AMReX.
